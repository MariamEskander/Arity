  • ᱜᱩᱱᱟᱹ : ᱟᱢ ᱫᱚ ᱜᱩᱱᱟᱹ ᱚᱯᱨᱮᱴᱚᱨ ᱵᱟᱝ ᱥᱯᱚᱥᱴ ᱡᱷᱚᱜ ᱵᱮᱵᱷᱟᱨ ᱫᱟᱲᱮᱭᱟᱜᱼᱟᱢ, ᱫᱟᱹᱭᱠᱟᱹ 3π+2(1+2)
  • ᱮᱠᱥᱯᱨᱮᱥᱚᱱ ᱠᱚᱱᱴᱤᱱᱩᱣᱮᱥᱚᱱ : starting a new expression with ᱢᱤᱫᱴᱟᱹᱝ ᱚᱯᱨᱮᱴᱚᱨᱥ (- ᱟᱲᱟᱜ ᱠᱟᱛᱮ)ᱥᱟᱶᱛᱮ ᱢᱤᱫᱴᱟᱹᱝ ᱱᱚᱣᱟ ᱮᱠᱥᱴᱮᱱᱥᱚᱱ ᱮᱛᱦᱚᱵ ᱟ ᱫᱚ ans ᱟᱡ ᱛᱮ ᱠᱷᱚᱧᱡᱟᱭᱟ, ᱢᱩᱪᱟᱹᱫ ᱮᱠᱥᱴᱮᱱᱥᱚᱱᱟᱜ ᱛᱮᱞᱟ ᱾
  • ᱥᱟᱯᱷᱟ : Enter ᱪᱟᱹᱵᱤ ᱵᱮᱵᱷᱟᱨ ᱠᱟᱛᱮ ᱯᱩᱨᱟ ᱚᱞ ᱞᱟᱹᱭᱤᱱ ᱵᱮᱵᱷᱟᱨ ᱢᱮ ᱾
  • ᱥᱟᱭᱱᱴᱤᱯᱷᱤᱠ ᱮ ᱱᱚᱴᱮᱥᱚᱱ : 1e3 ᱫᱚ 1000 ᱠᱟᱱᱟ ᱾
  • ᱨᱮᱰᱤᱭᱟᱱᱥ ᱵᱟᱝ ᱠᱟᱛᱮ ᱰᱤᱜᱽᱨᱤ ᱨᱮ ᱠᱩᱬ : sind(90) ᱥᱮ sin(90deg) ᱵᱮᱵᱷᱟᱨ ᱢᱮ ᱾
  • ᱱᱟᱜᱟᱢ ᱥᱮᱫ ᱪᱟᱞᱟᱜ ᱞᱟᱹᱜᱤᱫ ᱴᱨᱟᱠᱵᱚᱞ ᱪᱮᱛᱟᱱ/ᱞᱟᱛᱟᱨ ᱵᱮᱵᱷᱟᱨ ᱢᱮ ᱾
  • ᱢᱤᱫᱴᱟᱹᱝ ᱱᱟᱜᱟᱢ ᱡᱤᱱᱤᱥ ᱵᱟᱪᱷᱟᱣ ᱢᱮ : ᱡᱤᱱᱤᱥ ᱨᱮ ᱰᱷᱮᱨ ᱜᱷᱟᱹᱬᱤᱡ ᱚᱛᱟᱭ ᱢᱮ ᱾

ᱰᱮᱨᱤᱵᱷᱮᱴᱤᱵᱷ

ᱯᱩᱭᱞᱩ ᱰᱮᱨᱤᱵᱷᱮᱴᱤᱵᱷ ᱫᱚ ᱢᱤᱫ ᱯᱷᱚᱝᱥᱚᱱᱟᱜ ᱯᱩᱭᱞᱩ ᱰᱮᱨᱤᱵᱷᱮᱴᱤᱵᱷ ᱢᱤᱫᱴᱟᱹᱝ ᱟᱨᱜᱭᱩᱢᱮᱱᱴ ᱥᱟᱶ ᱯᱨᱟᱭᱤᱢ ᱟᱨᱜᱭᱩᱢᱮᱱᱴ ᱛᱮ ᱛᱮᱞᱟ ᱚᱰᱚᱠ ᱜᱟᱱᱚᱜᱼᱟ ᱺ log'(5) ᱾

ᱯᱷᱚᱝᱥᱚᱱᱟᱜ ᱧᱩᱛᱩᱢ ᱛᱟᱭᱚᱢ ᱯᱨᱟᱭᱤᱢ ᱪᱤᱱᱦᱟᱹ (ᱠᱚᱴ) ᱱᱤᱦᱟᱹᱛᱤ ᱫᱮᱠᱷᱟᱣᱜ ᱠᱟᱛᱷᱟ, ᱟᱨ ᱤᱱᱟᱹ ᱛᱟᱭᱚᱢ ᱜᱷᱮᱨᱟᱣ ᱛᱟᱦᱮᱸᱱ ᱢᱟᱸ ᱾

ᱰᱮᱨᱤᱵᱷᱮᱴᱤᱵᱷ ᱫᱚ ᱯᱞᱚᱴ ᱜᱟᱱᱚᱜᱼᱟ ᱾ ᱫᱟᱹᱭᱠᱟᱹ ᱺ sqrt'(x) ᱾

ᱢᱤᱫᱴᱟᱹᱝ ᱮᱯᱥᱯᱨᱮᱥᱚᱱᱟᱜ ᱰᱮᱨᱤᱵᱷᱮᱴᱤᱵᱷ ᱚᱰᱚᱠ ᱞᱟᱹᱜᱤᱫ ᱟᱢ ᱫᱚ ᱢᱤᱫᱴᱟᱹᱝ ᱮᱠᱥᱴᱮᱱᱥᱚᱱ ᱧᱩᱛᱩᱢ ᱯᱷᱚᱝᱥᱚᱱ ᱞᱮᱠᱷᱟ ᱵᱟᱪᱟᱣ ᱛᱮ ᱦᱩᱭᱟᱢᱟ : ᱫᱟᱹᱭᱠᱟᱹ ᱺ f(x)=x^3+x^2+1, ᱯᱟᱧᱡᱟ ᱛᱮ f'(x) ᱾

ᱟᱭᱢᱟᱸ ᱯᱞᱚᱴ

ᱚᱱᱟ 2d ᱜᱽᱨᱟᱯᱷ ᱨᱮ ᱟᱭᱢᱟᱸ ᱯᱷᱚᱝᱥᱚᱱ ᱯᱞᱚᱴ ᱞᱟᱹᱜᱤᱫ, ᱠᱷᱟᱹᱞᱤ ᱚᱱᱟᱠᱚ ᱚᱱᱟ ᱞᱟᱹᱭᱤᱱ ᱨᱮ ";" ᱵᱮᱵᱷᱟᱨ ᱟᱛᱮ ᱟᱫᱮᱨ ᱢᱮ ᱾ ᱫᱟᱹᱭᱠᱟᱹ ᱺ x;x^2;2

ᱵᱟᱭᱱᱟᱨᱤ, ᱚᱠᱴᱟᱞ, ᱦᱮᱠᱥᱟᱰᱮᱥᱤᱢᱟᱞ

ᱟᱢ ᱫᱚ ᱵᱟᱭᱱᱟᱨᱤ, ᱚᱠᱴᱟᱞ ᱥᱮ ᱦᱮᱠᱥᱟᱰᱮᱥᱤᱢᱟᱞ ᱨᱮ 0b, 0o or 0x ᱢᱟᱲᱟᱝ ᱴᱷᱮᱞ ᱵᱮᱵᱷᱟᱨ ᱫᱟᱲᱮᱭᱟᱜᱼᱟ, ᱡᱮᱞᱠᱟ:
  • ᱵᱟᱭᱱᱟᱨᱤ: 0b1010
  • ᱚᱠᱴᱟᱞ: 0o17
  • ᱦᱮᱠᱥᱟ: 0x100
ᱱᱤᱛᱚᱨ ᱚᱱᱟ ᱠᱟᱹᱢᱤ ᱩᱞᱴᱟᱹ ᱵᱟᱭ ᱜᱟᱱᱚᱜᱼᱟ, ᱫᱟᱹᱭᱠᱟᱹ ᱺ ᱵᱤᱱᱼᱰᱮᱥᱤᱢᱟᱞ ᱵᱮᱥ ᱨᱮ ᱛᱟᱞᱟ ᱮᱢᱚᱜᱼᱟ ᱾
